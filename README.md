# Python resume generator

## Workflow

1. Run `python watch_resume.py` in a terminal.
2. Open `resume.yaml` in a text editor.
3. Open `resume.pdf` in a PDF viewer (Preview on Mac, Adobe, SumatraPDF on Windows, etc.).
4. Edit and save `resume.yaml`. The PDF will auto-regenerate and the viewer will auto-refresh.
5. Export `resume.html` and `resume.pdf`.
